Two Architectures We Will Build
Baseline A End-to-End Faster R-CNN
1
ResNet-50 + FPN Backbone
Feature pyramid extracts multi-scale representations from the full blood smear
2
Region Proposal Network (RPN)
Proposes candidate cell bounding boxes at multiple anchor scales
3
RoI Align + Classification Head
Refines boxes, predicts class: RBC vs infected (trophozoite, ring, etc.)
4
mAP Evaluation (COCO metric)
IoU thresholds 0.5–0.95, per-class AP, recall curve
Novel B Decoupled Two-Stage Pipeline
1
Universal Cell Segmentation
Watershed + distance transform finds EVERY cell — infected or not, annotated or not
2
Cell Crop Extraction
Each detected cell → padded 64×64 crop. Map ground-truth labels via IoU overlap
3
EfficientNet-B0 Classifier
Lightweight CNN trained only on single-cell crops. Focal loss for class imbalance
4
Grad-CAM++ Explainability
Attention heatmap pinpoints the parasite inside the cell — clinical validation layer
